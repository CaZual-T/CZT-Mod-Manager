- Download [czt_nxs_bridge.rar](https://github.com/CaZual-T/CZT-Mod-Manager/blob/main/CZT_Extension/czt_nxs_bridge.rar)
- Extract to a location of your choosing (recommend putting it in the CZT Mod Manager/plugins folder.)
- Open your browser > Manage Extensions > Enable Dev Mode > Load Unpacked (select czt_nxz_bridge folder)
   - Once the extension is installed, Pin it to the top and then select the "CZT protocol" checkbox.

     <img width="357" height="260" alt="image" src="https://github.com/user-attachments/assets/841534c6-6bca-4abb-a775-e52c7d1bbcd4" />

   - Click Setup
   - Go to https://www.nexusmods.com/settings/api-keys and scroll down to CZT Mod Manager
 
      <img width="627" height="179" alt="image" src="https://github.com/user-attachments/assets/23ba92b8-bad0-4abb-9215-4b4af760f35b" />

   - Click 'Request API Key'
   - Copy and Paste into extension.

     <img width="447" height="211" alt="image" src="https://github.com/user-attachments/assets/150d3e80-383f-4a87-99c1-37864ac10c42" />

   - Go to nexus and download a mod using the manual download button to trigger CZT extension.
     
     <img width="358" height="57" alt="image" src="https://github.com/user-attachments/assets/f3062084-1d5d-4449-a917-8f8e3c0480a7" /></br>
     <img width="573" height="200" alt="image" src="https://github.com/user-attachments/assets/7c1bd086-48d2-4b9e-bdc3-47d1c05911aa" />

      - Free accounts: CZT will trigger after you click 'slow download'
      - Premium accounts: Triggers once you click 'manual download'
      - Note: if the mod manager download button is available. Both account types can use that without this extension.
   - Once your download is complete, CZT will begin installing the file automatically.

     <img width="1111" height="613" alt="image" src="https://github.com/user-attachments/assets/6a06b2a0-6788-4c7d-92a8-868099435b19" />

Notes:
- This is not for "Mod Manager Download" button. CZT already handles that in-app.
- This is for when theres no manager button and only the manual download exists.

Tested On:
- Opera GX, Chrome, MS Edge.
- Premium and Free nexus acccounts.
