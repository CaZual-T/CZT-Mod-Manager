## Main Note(s):
- CZT Mod Manager handles the mod files themselves, but doesnt 'load' anything into the game on its own.
- Manager Tasks (how it handles mods): install/uninstall, enable/disable, stage, symlink, load order etc.</br>
"Why does this matter?" ⬇️
  - ➡️ it matters because you still need to have melon loader installed.
  - Melon loader is what actual 'loads' a mod from your mod folder into the game itself.
    - https://github.com/LavaGang/MelonLoader/releases/download/v0.7.3/MelonLoader.Installer.exe
