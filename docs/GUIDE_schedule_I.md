## Main Note(s):
- CZT Mod Manager handles the mod files themselves, but doesnt 'load' anything into the game on its own.
- Manager Tasks (how it handles mods): install/uninstall, enable/disable, stage, symlink, load order etc.
##
- You still need to have melon loader installed
- Melon loader is what actual 'loads' the mod from your mod folder.
