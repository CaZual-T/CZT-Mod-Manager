# Install
1: Download UTM CZT Plugin.rar (extract folder to desktop, or wherever) </br>
2: Put the UTM_plugin FOLDER in CZT Mod Manager/plugins/scripts </br>

# how to use
- 1: Install mods like usual (drag and drop into log box, select, install)
- 2: Press F3 to open the plugin manager
    - 1st time use setup > ensure 'run at startup' is unchecked> save.
- 3: Run UTM
    - The UTM cmd window will appear.
- 4: Once completed, go to the manage tab. data7.pak is your merged file. Delete the old mod files. (done)

> [!TIP]
> - You can hover over the display names to see true file names. Useful if data7 already has saved "mod info" within the mod_list.json</br>
> - You can also shift left click a mod to change its display name.

> [!NOTE]
> - If you are already in the manage tab at the time of merge you will not see data7 until you reload the ui.mod list. </br>
>     - You can do this by simply switching to any other tab then going back to the manage tab.
